人海交错 总有因果

Mar 2, 2019 | 胡言乱语

我不满意自己，心里感到难受，为自己的生活惋惜。生活过得如此之快，而且没有意思。我老是在想，我的心那么难受，能把它从胸膛里掏出来就好了。

最近又像圣诞那段时间一样，吃不下睡不着，长期处于半低血糖的状态下，昏昏沉沉浑浑噩噩。不过没关系，还有那么多燃烧生命的人，我们这些情感接收者，是一定会被感动的。

很想看一出音乐剧或是舞台剧。

以前我说过追星女孩的情感浓度高到难以置信，虽然不是追星，但是现下的疯狂也是一样的道理。这一现象在听到合唱的“如何劝说自己没有你也可以”的时候达到了巅峰。可能过很久都会记得，我看完采访稿的感觉，如果理想主义注定被辜负，那么一起被辜负也是一种至高无上的浪漫。

很多人试图把情感投影在别人存在的平面上，去找寻现实生活中不存在的情谊，所谓的“借来的快乐也是快乐”。我一直认为，去做疯狂的事，尤其是探索艺术，拥有天赋和勇气不稀奇，一定还要有同等痛度的血肉病痛，生存的挣扎折磨和精神上的崩溃游离。俗一点就是“文章憎命达”绝不只是一句无病呻吟而已。然而在这样一条路上，竟然也能获得一份真正的理解、信任，一份可以背后high five的默契，如此Soulmate怎么感慨呢…就是“不止一次错把满地月光当海洋”这样的迷离和欲罢不能吧。

可能人生的Lucky值是一定的，要有lowest low才能有highest high，然而我们这些屏幕前的普通人呢，掩盖在城市楼房人群里，我们不恰巧地把幸运值均分在漫长的人生里，平淡无奇的生活里，多的只能是俗气的小确幸和丧气。这样的话，不汲取想象中的感情又怎么活下去呢。

最近看完了网飞的Sex education，感觉是一部非常小清新内核的性教育电视剧。一开始是因为Otis这段话去看的。事实上这一段并不出彩，我最感兴趣的甚至不是他这个男主角。
